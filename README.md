# Finance Markets Dashboard

A modern, responsive financial markets dashboard built with React, Tailwind CSS, and the Base44 SDK.

## Features
- **Dashboard**: Real-time market overview, news, and portfolio snapshots.
- **Screener**: Advanced stock and ETF screening with custom filters and technical signals.
- **Asset Analysis**: Detailed charts, metrics, and analyst insights for individual tickers.
- **Gemini AI Insights**: Automated market analysis powered by Gemini 3 Flash.
- **Responsive Design**: optimized for desktop, tablet, and mobile views.

## Tech Stack
- React
- Vite
- Tailwind CSS
- Recharts
- Base44 SDK (LLM and Auth integrations)

## Getting Started
1. Install dependencies: `npm install`
2. Run locally: `npm run dev`
